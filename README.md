# chat

This is a simple chat application built in React Native

Build and Run the app:

    Install React Native as described at https://facebook.github.io/react-native/docs/getting-started.html#content and https://reactnative.dev/docs/environment-setup
    Clone this repository.
    cd into the project
    Run npm-install to install all required android components.
    Run react-native run-android.
